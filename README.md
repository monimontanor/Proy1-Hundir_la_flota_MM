# 🚢 Hundir la Flota - Proyecto Python

## 🎯 Descripción

Este proyecto es una primera versión del juego Hundir la Flota desarrollado en Python utilizando NumPy.

El objetivo es simular la lógica básica del juego mediante la creación de un tablero, la colocación de barcos y la función de disparo.

---

## 🧩 Funcionalidades implementadas

- Creación de un tablero 10x10
- Colocación manual de barcos en el tablero a partir de listas predeterminadas
- Generación de barcos con coordenadas aleatorias (según eslora)
- Solicitud de coordenadas al usuario
- Sistema básico de disparo:
  - Impacto en barco
  - Agua
  - Control de casillas ya disparadas

---

## 🛠️ Tecnologías utilizadas

- Python
- NumPy
